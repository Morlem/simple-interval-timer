Simple-Interval-Timer

  • Responsive interval timer you can add to any web page
  • Customizable prepare, work, and rest times
  • Additional settings (see customized timer below)

Download

simple-interval-timer.zip

GitHub for source: https://github.com/Morlem/simple-interval-timer

Customized Timer


<head>
    ...
    ...
    <link rel="stylesheet" href="PATH-TO-CSS/simple-interval-timer.css">
</head>
<body>
    <div class="insert-custom-timer-here"></div>
    ...
    ...
    <script src="PATH-TO-SCRIPT/simple-interval-timer.min.js" charset="utf-8"></script>
    <script>
        (function() {
            simpleIntervalTimer({
                containerClass: "insert-custom-timer-here",
                prepareSeconds: 0,
                workSeconds: 10,
                restSeconds: 5,
                rewindSeconds: 5,
                editable: false,
                playSounds: false,
                showStopButton: false,
            });
        })();
    </script>
</body>
